# aliases: gif

```bash
@gif \
	[~download,dryrun,~upload] \
	[.|<object-name>] \
	[--frame_duration <150>] \
	[--output_filename <object-name>.gif] \
	[--scale <1>] \
	[--suffix <.png>]
 . generate <object-name>.gif.
@gif \
	open \
	[download,filename=<filename.gif>] \
	[.|<object-name>]
 . open <object-name>.gif.
```
